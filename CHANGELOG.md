# Changelog

All notable changes to this project are documented in this file.

## [1.0.0] - 2026-02-23

### Added

- Auto-copy `file://` URL for newest screenshot to clipboard.
- `launchd` agent installation script (`scripts/install.sh`).
- Uninstall script (`scripts/uninstall.sh`).
- Example launch agent plist.
- Project docs: README, ROADMAP, CONTRIBUTING, MIT license.
- GitHub Actions CI for shell script linting with `shellcheck`.
